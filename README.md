===Gruppuppgift Grupp 4===

Använd sifferinmatning för att navigera menyerna.
Alla textfiler ligger i programmets bin-mapp.

Vi som har ansvarat för de olika delarna är:
Sarisa - Användargränssnitt.
Simon - Metoder för att skapa och hantera lokaler.
John - Metoder för att skapa och hantera bokningar.
Gustav - Filhantering och korrektur av kod (Senior Developer).

Kända begränsningar:
Det går inte att boka samma lokal på flera olika tider i förväg, det går bara att boka en lokal om bokningen tagits bort eller om slutdatum passerat.
Ingen realtime-uppdatering.
Variabelnamnen är på Svenska, Engelska och Svengelska. Kanske fler språk, vid det här laget vågar ingen av oss kolla längre.
